About this site

A small static site where I keep notes and visualizations of things I'm learning.

This is a learning notebook — a place to write things down so I actually understand them, and to keep the interactive diagrams I build along the way.

What's here

Notes are short writings — closer to journal entries than blog posts. They live in posts/.

Visualizations are self-contained HTML pages, often with a lot of custom CSS and a little JavaScript. They live in visualizations/.

How it's built

Plain HTML, CSS, and JavaScript — no build step, no framework, no tracking. Hosted on GitHub Pages. A single posts.js manifest powers the home page so adding content is a one-file edit.
